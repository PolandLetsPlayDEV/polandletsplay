What is the API?

This public API lets you access live information, project data, and updates from Engineering Softcodes™. Perfect for tools, apps, dashboards, or if you want to show live patch notes and more.

Example Endpoints

GET /api/projects
GET /api/insane-forest
GET /api/latest-news
POST /api/bug-report

Authentication

Most endpoints are currently open. Future versions may require an API key for advanced or sensitive data.

Authorization: Bearer YOUR_API_KEY

Rate Limits

Please limit to 60 requests per minute. Abuse will result in IP bans.

How to Use

You can use tools like Postman or simple JavaScript/Fetch to connect with the API.
